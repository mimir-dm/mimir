## Encounter Multipliers

| # Monsters | Multiplier |
|------------|------------|
| 1 | x1 |
| 2 | x1.5 |
| 3-6 | x2 |
| 7-10 | x2.5 |
| 11-14 | x3 |
| 15+ | x4 |

For parties smaller than 3, use the next higher multiplier. For parties larger than 5, use the next lower multiplier.
