# Module Types

When creating modules with `create_module`, use these types for appropriate document templates:

## Available Types

### mystery
Investigation-focused adventures. Templates include:
- Clues and evidence tracking
- Suspect lists
- Timeline of events
- Red herrings

### dungeon
Classic dungeon crawls. Templates include:
- Room-by-room breakdown
- Encounter areas
- Treasure placement
- Environmental hazards

### heist
Stealth and planning adventures. Templates include:
- Target information
- Security measures
- Entry/exit points
- Contingency plans

### horror
Dark, atmospheric scenarios. Templates include:
- Tension building elements
- Horror beats
- Safe moments
- Escalation timeline

### political
Intrigue and social encounters. Templates include:
- Faction relationships
- Key figures
- Political stakes
- Social encounter guides

## Choosing a Module Type

Consider:
- **Primary gameplay**: Combat-heavy → dungeon, Social-heavy → political
- **Pacing**: Fast action → dungeon/heist, Slow burn → mystery/horror
- **Player agency**: High planning → heist, High investigation → mystery
- **Tone**: Light → dungeon, Dark → horror, Complex → political
