# Documentation Overhaul Initiative

## Context

Documentation has fallen out of sync with the codebase after significant feature additions. Analysis revealed:
- Critical errors (React/Vue mismatch)
- Major undocumented features (Character/NPC, Books, PDF export)
- Outdated technical details (migration counts, table counts)

## Goals & Non-Goals

**Goals:**
- Fix all critical documentation errors
- Document major user-facing features
- Update technical architecture docs to match current state
- Ensure new developers can onboard effectively

**Non-Goals:**
- API reference documentation (future work)
- Video tutorials or interactive guides
