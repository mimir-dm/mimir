# PDF Template Redesign Initiative

## Context

Redesign all PDF export templates to be polished, print-friendly, and aligned with WotC-style D&D conventions. Each template gets a dedicated design task to ensure quality.

## Goals
- WotC 2-page character sheet as the default export
- Every PDF template reviewed and improved
- Standard actions reference on character sheets
- Spell slot tables (max only, no tracking)
- All 18 skills with computed bonuses

## Non-Goals
- Tracking consumables (HP, spell slots used, ammo) — players use paper
- Spell name lists on character sheet (separate spell cards export exists)

## Templates to Redesign

1. **Character Sheet** — WotC-style 2-page (stats + backstory/reference)
2. **Character Battle Card** — half-page combat quick-reference
3. **Character Long Form** — extended roleplay/narrative
4. **Spell Cards** — per-class spell list cards
5. **Equipment Cards** — magic item reference cards
6. **Equipment Detail** — inventory list
7. **Monster Cards** — stat block cards
8. **Trap Cards** — trap reference cards
9. **Module Documents** — campaign docs with monsters/traps/maps
10. **Campaign Documents** — campaign-level export
11. **Map Print** — preview and tiled map output
