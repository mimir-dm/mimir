## Content Mapping (from character markdown)

| Section | Source Field |
|---------|--------------|
| **Appearance** | `## APPEARANCE & PERSONALITY` physical description |
| **Distinctive Features** | Unique visual traits (frost patterns, special equipment) |
| **Personality Traits** | `### Personality Traits` |
| **Ideals** | `### Ideals` |
| **Bonds** | `### Bonds` |
| **Flaws** | `### Flaws` |
| **Background Type** | `## BACKGROUND: <TYPE>` |
| **Feature** | `### Feature: <name>` |
| **Backstory** | Background narrative text |
| **Voice & Mannerisms** | From `## ROLEPLAYING NOTES` |
| **Key Relationships** | Party members, NPCs, organizations |
| **Character Goals** | Current objectives, long-term ambitions |
| **Play Reminders** | Bullet points from `## ROLEPLAYING NOTES` |

## Notes

- This is a single-page layout, may extend to 2 pages if backstory/notes are lengthy
- Backstory box should auto-expand based on content
- Play Reminders are bullet-pointed quick reference for at-table RP
- Portrait is NOT included here (it's on compact sheet page 2)
