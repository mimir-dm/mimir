## Columns

| Column | Content |
|--------|---------|
| Item | Equipment name |
| Weight | Weight in lbs (if known) |
| Properties / Notes | Damage, AC, special abilities, quantities, whatever is relevant |

## Notes

- Flat list - no categorization needed
- Page expands based on inventory size
- Items with fluff/special rules get full-width description block below the header row
- Becomes vertical card-like layout for special items, compact rows for mundane gear
- Currency at top for quick reference
- Total weight at bottom
